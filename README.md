            <p>The <span class="highlight">E Store</span> is a cutting-edge e-commerce website that offers a wide range of electronics for sale.

            Apple, Samsung, and others. The goal is to provide users with an intuitive, mobile-responsive, and user-friendly shopping experience.</p>
        </div>

        <div class="section">
            <h2>Features</h2>
            <ul>
                <li>Product catalog with detailed descriptions and pricing</li>
                <li>User-friendly navigation for browsing categories</li>
                <li>Filter options for easy searching by product type, brand, price, etc.</li>
                <li>Shopping cart for product selection and checkout</li>
                <li>Responsive design optimized for both mobile and desktop views</li>
                <li>Secure payment gateway integration (future implementation)</li>
                <li>Figma-designed user interface for modern aesthetics</li>
            </ul>
